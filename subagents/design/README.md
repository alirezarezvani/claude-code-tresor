## Overview

The Design team agents provide comprehensive expertise in user experience, user interface design, visual design, and brand identity. These agents help create beautiful, functional, and delightful user experiences that drive engagement and satisfaction.

### Team Specializations

- **UI Design** - Interface design, design systems, component libraries
- **UX Research** - User research, journey mapping, behavior analysis
- **Visual Design** - Visual storytelling, infographics, presentations
- **Brand Identity** - Brand guidelines, consistency, asset management

## Standards Integration

All design agents follow standards from `/standards/` folder:
- **Design Systems** - Component libraries, design tokens, style guides
- **Accessibility** - WCAG 2.1 AA compliance, inclusive design principles
- **Brand Guidelines** - Visual identity, voice and tone, asset usage
- **UX Best Practices** - User-centered design, usability principles
- **Responsive Design** - Mobile-first, adaptive layouts, progressive enhancement
